Mnémlet

Brain-inspired memory for your self-hosted AI stack

Runs on a Pi. No cloud. No API keys. Your data stays on your hardware.

Forgets Intelligently

Exponential decay + interaction-weighting. What you use stays sharp. What you ignore fades. No manual cleanup.

Thinks While You Sleep

Night consolidation: dedup, rescore, cluster, morning briefing. All local, zero API costs. Like REM sleep for your agent.

Inspectable Vault

Every memory as a Markdown file. cat ~/.mnemlet/vault/**/*.md. Open in Obsidian. No black box.

See It Work

Full ingest → recall → sleep → vault cycle from the terminal.

~/mnemlet — bash 
$ curl -X POST http://localhost:4050/api/v1/ingest \
  -H 'Content-Type: application/json' \
  -d '{"content":"Christoph prefers dark mode","namespace":"preferences"}'
{"memory_id":"01d0e...","stored":true,"retention_score":0.45}

$ curl -X POST http://localhost:4050/api/v1/recall \
  -H 'Content-Type: application/json' \
  -d '{"query":"editor theme preference"}'
{"results":[{"content":"Christoph prefers dark mode in all editors","score":0.28}]}

$ curl -X POST http://localhost:4050/api/v1/sleep/start
[sleep] Running: _task_dedup_today
{"status":"started","checkpoint":{"_task_dedup_today":true}}

$ cat ~/.mnemlet/vault/preferences/2026-05/01d0e....md
---
id: 01d0e...
namespace: preferences
retention_score: 0.45
---
Christoph prefers dark mode in all editors

If your priority is cloud sync, a polished dashboard, or an ecosystem with 50k stars — use Mem0 or MemPalace. If your priority is local-first, brain-inspired forgetting, and running on hardware you own — that's Mnémlet.

Quickstart

Three commands. That's it.

~/mnemlet — bash 
$ pip install mnemlet

$ mnemlet serve

$ curl -X POST http://localhost:4050/api/v1/ingest \
  -H 'Content-Type: application/json' \
  -d '{"content":"Hello Mnémlet","namespace":"test"}'
