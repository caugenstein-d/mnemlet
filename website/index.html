Forgets Intelligently

Exponential decay + interaction-weighting. Preferences last years. Daily chat fades in days. What you use stays sharp. What you ignore disappears. No manual cleanup.

Thinks While You Sleep

After 2h inactivity: dedup, rescore, cluster, morning briefing. Runs entirely on-device. Zero API calls. Like REM sleep for your agents.

Inspectable Vault

Every memory as a .md file with YAML frontmatter. Open in Obsidian. grep it. git it. No black box database lock-in.

See It Work

Full ingest → recall → sleep → vault cycle. Real output from a Raspberry Pi.

demo — bash 
$  curl POST /api/v1/ingest -d '{"content":"Christoph prefers dark mode","namespace":"preferences"}'
{"memory_id":"01d0e...","stored":true,"retention_score":0.45}

$  curl POST /api/v1/recall -d '{"query":"editor theme preference"}'
{"results":[{"content":"Christoph prefers dark mode in all editors","score":0.28}]}

$  curl POST /api/v1/sleep/start

{"status":"started","checkpoint":{"_task_dedup_today":true}}

Quickstart

Three commands. That's it.

~/mnemlet — bash 
$ pip install mnemlet

$ mnemlet serve
🧠 Mnémlet v0.1.0 — http://127.0.0.1:4050

$ curl -X POST http://localhost:4050/api/v1/ingest \
  -H 'Content-Type: application/json' \
  -d '{"content":"Hello Mnémlet","namespace":"test"}'
{"memory_id":"...","stored":true}
