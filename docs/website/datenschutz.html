Datenschutzerklärung

Der Schutz Ihrer Daten ist uns wichtig. Diese Datenschutzerklärung informiert Sie über den Umgang mit personenbezogenen Daten auf dieser Website.

Keine Erhebung personenbezogener Daten

Diese Website ist eine rein statische Projekt-Dokumentation. Es werden keine personenbezogenen Daten erhoben, verarbeitet oder gespeichert:

  • Keine Benutzerkonten oder Registrierungen
  • Keine Kontaktformulare
  • Keine Kommentarfunktionen
  • Keine Newsletter oder Benachrichtigungen

Beim Aufruf der Website können durch den Hosting-Anbieter (siehe unten) technisch bedingt Server-Logdateien erstellt werden, die unter anderem IP-Adressen enthalten können. Dies liegt außerhalb unseres Einflussbereichs und entspricht dem üblichen Betrieb von Webservern.

Keine Cookies

Diese Website setzt keine Cookies. Es werden keinerlei Daten im Browser des Besuchers gespeichert.

Keine Analyse-Tools; externe Schrift- und Skript-Ressourcen

Es kommen keine Webanalyse-Tools (wie Google Analytics, Matomo o. Ä.), keine Tracking-Pixel und keine Werbenetzwerke zum Einsatz. Für Gestaltung und Layout lädt die Website derzeit Google Fonts sowie Tailwind CSS über öffentliche CDN-Adressen. Dabei können technisch bedingt IP-Adresse, Browserdaten und Zeitpunkt des Abrufs an diese Anbieter übertragen werden. Mnémlet selbst erhebt darüber hinaus keine personenbezogenen Daten auf dieser Website.
